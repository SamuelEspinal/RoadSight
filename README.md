# RoadSight

RoadSight is a React + TypeScript + Mapbox GL road inspection dashboard built as a small demo project.

## Features
- Interactive Mapbox GL road visualization
- Condition-based filtering (Good / Fair / Poor)
- Click-to-select road segments
- Highlight + zoom to selected segment
- Inspection details sidebar with notes UI
- Statistics summary for visible road conditions

## Tech Stack
- React
- TypeScript
- Vite
- Mapbox GL JS

## Purpose
I built this project to get hands-on with Mapbox GL and TypeScript while exploring the kind of geospatial UI workflows used in road inspection / infrastructure applications.
